# Assignment 1: Rule-Based AI Agent

## Objective
Build a simple AI agent using rule-based logic.

## Approach
The agent follows a pipeline:
Input → Intent Detection → Action Execution

- Input handler processes user input
- Decision logic uses keyword matching
- Action execution performs tasks

## Installation
No external libraries required.

## Execution
Run:
python agent.py

## Example
Input: calculate 2 + 3  
Output: Result: 5

## Learning Outcomes
- Basic agent architecture
- Rule-based reasoning
- Modular design
